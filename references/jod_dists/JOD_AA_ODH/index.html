The Alto Adige Open Data Hub is a public OpenData provider that expose many different Dataset. This JOD AA ODH can handle all dataset from Traffic & Mobility.
Traffic & Mobility datasets provide their data in the form of station. Each station can be exposed as a JOSP Object that provide different statuses depending on his StationType.

User's can configure the JOD AA ODH distribution and choose station represent as JOSP Object. Edit the configs/configs.sh|ps1 file and update following properties JOD_AA_ODH_STATION_TYPE and JOD_AA_ODH_STATION_CODE.

In the below list, for each data set (in bold), are listed all available StationTypes. Click on dataset's link for a short description of station types and his data providers. Click station type's link to show a list of all station of that type and their scode.
