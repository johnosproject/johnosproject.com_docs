# Publish JOD Distribution

To publish your own JOD Distribution is enough to set execution mode to ```prod``` and execute the ```scripts/publish.sh|ps1``` script from [JOD Template Commands](https://bitbucket.org/johnosproject_shared/com.robypomper.josp.jod.template/src/develop/docs/tmpl/tmpl.md).

To set the execution mode to ```prod``` you must edit the ```JCP_ENV``` property in the ```configs/jod_dist_configs.sh|ps1``` file from [JOD Distribution configs](configuration/distribution_configs). Then you must set also ```JCP_ID``` and ```JCP_SECRET``` values.<br/>

```JCP_ID``` and ```JCP_SECRET``` values depends on local JCP instance.<br/>

:::info For ```JCP_ENV='prod'```, ```JCP_ID``` and ```JCP_SECRET``` values request your own credentials at [tech@johnosproject.com](mailto:tech@johnosproject.com).<br/>
:::

The ```scripts/publish.sh|ps1``` scripts generate compressed archives for your own JOD Distribution. You can distribute them to your friends or users, or publish them on the official [JOD Distributions list](/docs/references/jod_dists/). *To publish your own distribution on that list, please contact us at [tech@johnosproject.com](mailto:tech@johnosproject.com).*

Generated archives depends on [JOD Distribution configs](configuration/distribution_configs) you set for your own distribution:

```shell
build/publications/{DEST_ARTIFACT}-{DEST_VER}.tgz   // tar.gz format
build/publications/{DEST_ARTIFACT}-{DEST_VER}.zip   // zip format
```
