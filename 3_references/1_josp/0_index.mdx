## JCP Micro-Services

| JCP Micro Service | Description and default ports |
|---|---|
| [JCP Database](jcp/microservices/dbms/) | DBMS based on [MySQL](https://www.mysql.com/) listening on port ```8999```. |
| [JCP Auth](jcp/microservices/auth/) | Auth service provided by a configured [Keycloak](https://www.keycloak.org/) listening on port ```8998```. |
| [JCP APIs](jcp/microservices/apis/) | Public APIs for JOSP EcoSystem. Used by JOSP Objects and Services to register them self, connect to gateways, uploads events.... This service is listening on port ```9001```. |
| [JCP Gateways](jcp/microservices/gws/) | A GW service that dispatch messages from JOSP Objects to JOSP Services and viceversa for cloud communication. This service is listening on port ```9002```. |
| [JCP JSL Web Bridge](jcp/microservices/jslwb/) | HTTP API wrapper for the [JSL](jsl/) library. It allow clients, like JavaScript websites, to initialize a JSL session and access to the JOSP EcoSystem as JOSP Services. This service is listening on port ```9003```. |
| [JCP Front End](jcp/microservices/fe/) | The main JCP web interface. It provides the web dashboard for users to manage their objects. This service is listening on port ```9004```. |

## JOSP Internal Libraries

* [JOSP APIs](internal/josp_apis/): collection of definitions commons to all 3 main JOSP components.
* [JOSP Commons](internal/josp_commons/): set of utilities commons to all 3 main JOSP components.
* [JCP Commons](internal/jcp_commons/): collection of definitions commons to all JCP micro-services.
* [JCP Service](internal/jcp_service/): set of utilities commons to all JCP micro-services.
* [JCP DB](internal/jcp_db/): collection of definitions commons to all JCP DB's library.
* [JCP DB APIs](internal/jcp_db_apis/): DB's library for JCP APIs micro-service.
* [JCP DB GWs](internal/jcp_db_gws/): DB's library for JCP GWs micro-service.
* [JCP DB FE](internal/jcp_db_fe/): DB's library for JCP FE micro-service.
