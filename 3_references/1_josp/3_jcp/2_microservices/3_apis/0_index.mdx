import ReferenceIntro from '../../../../_references_intro.mdx'

<ReferenceIntro
  logo_path="JCP_APIs_Logo_250.png">

The **John Cloud Platform APIs is the APIs manager for a JOSP EcoSystem**.
All JOSP EcoSystem components can use the JCP APIs as a HTTP entry point to handle objects, services and users on the JCP.

The public JCP APIs can be reached at [https://api.johnosproject.org](https://api.johnosproject.org/swagger-ui.html) url.
</ReferenceIntro>
