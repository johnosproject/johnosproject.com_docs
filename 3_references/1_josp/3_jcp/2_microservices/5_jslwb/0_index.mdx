# JCP JSL Web Bridge Reference

import ReferenceIntro from '../../../../_references_intro.mdx'
import ReferenceInfo from '../../../../_references_info.mdx'

<ReferenceIntro
  logo_path="JCP_JSL_Web_Bridge_Logo_250.png">

The **John Cloud Platform JSL Web bridge is the HTTP APIs implementation of the JSL library for an entire JOSP EcoSystem**.
With the APIs exposed by this microservice, all API's client can create a JSL session and interact with JOSP Objects.<br/>
Each client (like a web site) is saw from the JOSP EcoSystem as a specific JOSP Service and not like JSL Web bridge services.

The public JCP APIs can be reached at [https://jslwb.johnosproject.org](https://jslwb.johnosproject.org/swagger-ui.html) url.
</ReferenceIntro>
