<ReferenceIntro
logo_path="JOSP_4_Google_Workspace_Logo_250.png">

The **JOSP 4 Google Workspace is an add-on for Google Workspaces that allow access to JOSP Objects from Google Documents like Spreadsheet**.
When enabled, this add-on, introduce new functions for Google Sheets formulas.
Those function allow you to list, filter and get object's values.
</ReferenceIntro>
