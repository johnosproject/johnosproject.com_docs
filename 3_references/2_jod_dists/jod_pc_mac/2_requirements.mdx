# Distribution's requirements

:::caution
This distribution can be installed only on MacOS. For other JOD PC distributions that support different OS, please see [JOD PC Linux](/docs/references/jod_dists/jod_pc_linux/) and [JOD PC Windows](/docs/references/jod_dists/jod_pc_win/).
:::

This distribution use an object structure that calls different scripts from ```scripts/hw``` directory.
Those scripts queries the host's OS to get object's statuses values. Some of them are dependant to external commands.

:::tip
For common JOD Distribution requirements, please see the [JOD Distribution TEMPLATE/How to use a JOD Distribution/Configure JOD Distribution](/docs/references/tools/jod_dist_template/how_to_use_a_jod_distribution/requirements) page.
:::

## Install ```top```

Command ```top``` is installed on all MacOS by default.
