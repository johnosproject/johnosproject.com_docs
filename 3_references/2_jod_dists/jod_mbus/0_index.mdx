<ReferenceIntro
  logo_path="JOD_MBus_Logo_250.png">

The __JOD MBus represent all devices connected to a MBus master devices as a JOSP Object__.

This distribution expose a single JOSP Object with Statues grouped by device connected to the MBus master.<br/>
This distribution communicate with the MBus master via ```mbus-serial-scan``` and ```mbus-serial-request-data``` commands from [```libmbus```](https://github.com/rscada/libmbus) tool.
So your MBus master must be available to those command to work with this distribution.
</ReferenceIntro>

**The JOD MBus distribution was developed and tested with the [M-Bus Master Hat](https://www.packom.net/m-bus-master-hat/) on a [Raspberry Pi](https://www.raspberrypi.org/).**<br/>
This is the reference HW for this distribution.
That means, you can still use this distribution with other MBus master (as long as your MBus master can be reachable from libmbus's tools), but this documentation is based on that HW configuration.<br/>
Here the [M-Bus Master Hat setup guide](https://www.packom.net/m-bus-master-hat-instructions/), follow it until 'Using libmbus'.
