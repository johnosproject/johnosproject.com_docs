import ReferenceIntro from '../../_references_intro.mdx'

<ReferenceIntro
  logo_path="JOD_PC_Linux_Logo_250.png">

The __JOD PC Linux allow to transform a Linux computer in a JOSP Object__.

This distribution is configured to use always the same object's structure. His structure is configured to use [Puller Shell](/docs/references/josp/jod/specs/workers/puller_shell) and [ShellExecutors](/docs/references/josp/jod/specs/workers/executor_shell) that executes custom scripts contained in the ```scripts/hw``` directory.
</ReferenceIntro>
