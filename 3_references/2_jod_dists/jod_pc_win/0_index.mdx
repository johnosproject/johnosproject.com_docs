# JOD PC Windows References

import ReferenceIntro from '../../_references_intro.mdx'
import ReferenceInfo from '../../_references_info.mdx'

<ReferenceIntro
  logo_path="JOD_PC_Windows_Logo_250.png">

The __JOD PC Windows allow to transform a Windows computer in a JOSP Object__.

This distribution is configured to starts background scripts contained in the ```scripts/hw/listeners``` directory before the JOD Agent startup.
Those scripts will query values to the Windows OS and write them to a temporary file. Then the object's structure is configured to use [Listener File](/docs/references/josp/jod/specs/workers/listener_file) and listen for temporary files changes.
</ReferenceIntro>
