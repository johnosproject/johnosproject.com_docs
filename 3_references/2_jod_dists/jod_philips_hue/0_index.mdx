import ReferenceIntro from '../../_references_intro.mdx'
import ReferenceInfo from '../../_references_info.mdx'

<ReferenceIntro
  logo_path="JOD_Philips_Hue_Logo_250.png">

The __JOD Philips Hue represent all devices connected to a Philips Hue Hub as a JOSP Object__.

This distribution expose a single JOSP Object with Actions and Statues grouped by light connected to the Philips Hub.<br/>
**This JOD distribution and the Philips Hue Hub must run on the same local network.**
That means the Philips Hub must be reachable via local network from the machine where we install and run this distribution.

This JOD Distribution interact with local Philips Hue hub via HTTPS requests.
</ReferenceIntro>
