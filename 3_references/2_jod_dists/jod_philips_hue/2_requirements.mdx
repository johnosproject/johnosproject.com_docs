# Distribution's requirements

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';
import CodeBlock from '@theme/CodeBlock';

:::caution
This distribution can be installed on Linux or MacOS.
:::

This distribution use the [```scripts/hw/generateObjectStructure.sh```](https://bitbucket.org/johnosproject_shared/com.robypomper.josp.jod.philips_hue/src/master/dists/resources/scripts/hw/generateObjectStructure.sh) script to generate the object structure on his startup.
This script require the ```curl``` and ```jq``` external commands to require and parse station's structure.

:::tip
For common JOD Distribution requirements, please see the [JOD Distribution TEMPLATE/How to use a JOD Distribution/Configure JOD Distribution](/docs/references/tools/jod_dist_template/how_to_use_a_jod_distribution/requirements) page.
:::

## Install ```jq```

You can download latest version from jq's [Download](https://stedolan.github.io/jq/download/) page or following below steps.
