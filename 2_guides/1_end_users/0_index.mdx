# End User's Guides

EndUser is intended as who is physically interacting with a connected object or who is using an IoT service.

Here, you can find all the guides relating to the daily use of objects and services, but also for their configuration and sharing.

_**For Developers:** We encourage the use of this HowTos inside your services as "Inline Help". Feel free to wrap this pages in your user interfaces, add their link or add them in the references._
