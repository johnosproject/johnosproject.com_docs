# Versions

The John O.S. Project provide multiple artifacts as software libraries, executables, distributions and so on. Because of that here are listed all versions for each artifact provided from the JOSP project.

---

## JOSP Components

This version group include all artifacts buildable from the John O.S. Project source code. It includes all JOSP Core and Internal components.

### JOSP 2.2.1

Released on **01/10/2021**.<br/>
Git Repository: [com.robypomper.josp:2.2.1](https://bitbucket.org/johnosproject_shared/com.robypomper.josp/src/2.2.1/)

Bug fix and improved component stability:

- Fixed generation bugs on JavaJKS from JOSP Commons
- Fixed missing instance termination when JCP authentication fails in JOD
- Improved support for Windows
- Updated PullerUnixShell and ExecutorUnixShell to PullerShell and ExecutorShell
- Other minor fixes

For a detailed changelist visit [JOSP-3](https://johnosproject.atlassian.net/browse/JOSP-3)
