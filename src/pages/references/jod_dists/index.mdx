# John Object Daemon Distributions

The [JOD Distribution TEMPLATE](/references/tools/jod_dist_template/) is the main tool to create a new JOD Distribution.<br/>
That's necessary only if you can't find a distribution that fits your needs on the bellow lists.

More distribution will be available soon, come back and check again.<br/>
To publish your own distribution on this list, please contact us at [tech@johnosproject.com](mailto:tech@johnosproject.com).
